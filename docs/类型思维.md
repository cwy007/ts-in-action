## what is ts ？

具有类型系统的js超级，可以编译成纯js

— 类型检查，ts 在编译代码时会进行严格的静态类型检查，在编码阶段发现可能存在的隐患

- 语言扩展，es6

- 工具属性，编译成 js，在各个平台运行

- vs code 自动补全，提示，使用接口定义代替文档，提高开发效率，降低维护成本
